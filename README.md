# プロジェクトの概要

## プロジェクト名
*株取引記録アプリ*

## プロジェクトの目的
株取引を記録し、反省できるようにする。  
初めての個人開発なので成果物として作成を行なった

## アプリ使用方法
これは、株式投資の記録アプリです。  
1.記録を書くで購入した株の情報を入力します。  
2.「現在の保有株一覧」で現在保有している株をみることができます。  
-----Done-----  
3.「株の売却」で保有している株の売却記録ができます。  
4.「損益一覧」で通算損益が見れます。  
5.「取引記録」でいままでの取引記録を表示できます。
